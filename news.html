二、各厂商2026上半年核心业绩与经营概况

1. 三星 Samsung（全球第1）

1. 营收表现：全球电视品类营收占行业总销售额31.3%，高端机型（1500美元以上）市占近50%，75寸以上超大尺寸营收份额31.6%； 2. 利润结构：QLED与QD-OLED高端产品贡献超70%电视业务利润，中小尺寸低端机型持续收缩； 3. 区域动作：正式退出中国大陆线下家电零售市场，集中资源北美、欧洲高端市场； 4. 亮点：QD-OLED出货同比+28.8%，OLED营收份额达40.1%，缩小与LG差距；
风险：存储芯片涨价抬升整机成本，低端出货承压，中国本土市场份额大幅萎缩。

2. TCL（全球第2）

1. 财报数据：2026Q1 TCL电子营收292.3亿港元（同比+15.3%），净利润3.8亿港元（同比+140%），盈利增速行业第一； 2. 增长驱动：北美、东南亚新兴市场放量，MiniLED高端机型均价提升，华星光电面板自供降低成本； 3. 战略优势：垂直整合面板+整机，大尺寸出货增速领跑前五，持续缩小与三星出货差距； 4. 渠道：海外自有品牌+代工双轮驱动，代工索尼电视订单增量显著。

3. 海信 Hisense（含东芝，全球第3）

1. 财报数据：海信视像2026Q1营收137.21亿元（+2.58%），归母净利润5.91亿元（+6.68%）； 2. 细分龙头：全球百寸及以上大屏销量份额52.9%，RGB-Mini LED国内渠道份额83%； 3. 事件催化：2026世界杯官方观赛设备供应商，海外市场收入占比超50%； 4. 品牌矩阵：海信主品牌+东芝高端影像双布局，欧洲、北美均价持续上涨。

4. LG 电子 LGE（全球第4）

1. 细分垄断：2026Q1全球WOLED电视出货市占50.5%，连续14年稳居OLED龙头； 2. 业绩情况：电视事业部Q1营收同比微增3.6%，依靠高端OLED拉动整体毛利转正； 3. 短板：LCD中端机型出货持续下滑，整体出货量被TCL、海信拉开差距； 4. 软件壁垒：webOS全球装机量稳步提升，平台增值服务收入持续增长。

5. 小米 Xiaomi（全球第5）

1. 出货下滑：上半年出货同比大幅下滑20.6%，核心原因全球各地家电以旧换新补贴退坡； 2. 产品调整：主动削减低毛利32寸入门机型，重心转向65寸以上中高端游戏电视； 3. 成本压力：存储、面板涨价挤压低端机型利润，海外外销规模收缩； 4. 优势：国内线上渠道性价比赛道仍保有稳定份额，米家IoT生态协同优势明显。

6. 创维 Skyworth（全球第6）

1. 增量来源：拿下飞利浦北美电视独家运营权、松下电视品牌授权，海外出货持续放量； 2. 产品结构：超薄壁画电视、MiniLED游戏电视提升均价，改善盈利水平； 3. 行业定位：国内线下渠道稳居第二，全球市场稳步冲击前五梯队； 4. 挑战：无自有面板厂，上游原材料波动对成本影响大于TCL、海信。

三、2026上半年全球电视行业整体总结

1. 总量规模：2026上半年全球电视整机出货约9420万台，同比增长3.2%；Q1淡季备货拉动增长，Q2受原材料涨价需求小幅走弱； 2. 格局变化：中国品牌（TCL+海信+创维）合计出货占比超35%，持续挤压韩系厂商份额；三星依旧总量第一，但领先优势持续收窄； 3. 技术趋势：MiniLED、超大屏75"+、OLED高端产品成为增长主力，中小尺寸低端机型全面收缩； 4. 成本压力：DRAM/NAND存储器涨价，中小电视生产成本上浮约8%-15%，全行业加速产品高端化转型。
